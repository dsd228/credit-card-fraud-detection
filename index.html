Credit Card Fraud Detection

Proyecto interdisciplinario de Machine Learning para detectar transacciones fraudulentas.
Dataset público de Kaggle. Resultados auditados y visualizados.

Ver código Ver notebook

Análisis Exploratorio

El dataset contiene 284,807 transacciones, de las cuales 492 son fraude (0.172%). El mayor reto es el desbalance de clases.

Algoritmos y Estrategia

Se probaron Random Forest, XGBoost, MLP y Regresión Logística. El balanceo se realizó con SMOTE.

Resultados

  • Accuracy: 99.95%
  • Recall: 89%
  • Precision: 92%
  • F1-Score: 0.90
  • ROC-AUC: 0.97
